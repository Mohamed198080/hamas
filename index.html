الأعمدة المطلوبة: رقم_العداد (نص)، الطول (رقم، خط الطول)، العرض (رقم، خط العرض)، الاجراء (نص: فتح عداد، قفل عداد، قفل طلب عميل، وضع ملصق)
يحتوي على مضلعات (Polygon أو MultiPolygon) بأسماء

⚠️ بعد تحميل الملف، قم برفعه إلى مستودع GitHub (بنفس مجلد index.html) باسم data.json
ثم أرسل روابط الفرق التالية لأفراد الفريق.

📌 تعليمات رفع الملف إلى GitHub:
1. افتح مستودعك على GitHub → Add file → Upload files.
2. اختر ملف data.json الذي حمّلته.
3. اكتب رسالة commit ثم Commit changes.
4. بعد دقيقة، ستعمل الروابط أدناه تلقائياً.
